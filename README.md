# Student Study Hub
A team-built web platform for university students, providing a central place to access subject resources, get support, and collaborate.

## Features
-**index** - home page and website info
- **Subjects** – browse resources and materials organised by subject
- **Forum** – discussion area for students to interact
- **Whiteboard** – collaborative space for notes or discussion
- **Support** – access help and guidance pages

## Tech Stack
- HTML
- CSS
- JavaScript
